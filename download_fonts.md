# Download Custom Fonts for Sparely

## Instructions:

1. **Download Poppins font:**
   - Go to: https://fonts.google.com/specimen/Poppins
   - Click "Download family"
   - Extract the ZIP file
   - Copy these files to `app/src/main/res/font/`:
     - `Poppins-Regular.ttf` → rename to `poppins_regular.ttf`
     - `Poppins-Medium.ttf` → rename to `poppins_medium.ttf`
     - `Poppins-SemiBold.ttf` → rename to `poppins_semibold.ttf`
     - `Poppins-Bold.ttf` → rename to `poppins_bold.ttf`
     - `Poppins-ExtraBold.ttf` → rename to `poppins_extrabold.ttf`

2. **Download Roboto Flex font:**
   - Go to: https://fonts.google.com/specimen/Roboto+Flex
   - Click "Download family"
   - Extract the ZIP file
   - Use the static fonts from the "static" folder
   - Copy these files to `app/src/main/res/font/`:
     - `RobotoFlex-Regular.ttf` → rename to `roboto_flex_regular.ttf`
     - `RobotoFlex-Medium.ttf` → rename to `roboto_flex_medium.ttf`
     - `RobotoFlex-SemiBold.ttf` → rename to `roboto_flex_semibold.ttf`
     - `RobotoFlex-Bold.ttf` → rename to `roboto_flex_bold.ttf`

## Important Notes:
- Font file names MUST be lowercase
- Use underscores (_) not hyphens (-)
- Use .ttf format
- Delete all the .xml files currently in the font folder
- Keep only the .ttf files
